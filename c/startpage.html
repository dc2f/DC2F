Start Page

DC2F

DC2F (Distributed Content Creation Framework) is a Content Management System which builds upon the idea to manage content the same way as code - with Git ;-)

Currently it is nothing more than a (very) simple Java Framework for a static page generator, quite similar to awestruct or jekyll.

Goal

The goal is to have a frontend editable website which can be hosted and edited distributed by having a git backend. Migrations, staging, merging, branching - everything you ever wanted to do in a CMS with the power of Git.
